# Taxhund: 8-Bit Tax Barrage

Vertical bullet hell shooter prototype built with plain HTML5 Canvas + JavaScript modules, now styled with a 16-bit SNES-inspired presentation using custom sprite assets.

## Run locally

Open `index.html` in a browser, or serve it as static files:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Set source to **Deploy from a branch**.
4. Select branch (`main` or `master`) and folder (`/root`).
5. Save and wait for the Pages URL.

No build step is required.

## Controls

- `WASD`: move
- `SPACE`: shoot
- `E`: bomb
- `ESC` / `P`: pause
- Title screen: `SPACE` or `ENTER` to start
- Game over/results: `R` retry, `T` title
